Dashboard interactivo de análisis geotérmico y vulnerabilidad económica para los 46 municipios del área de influencia del Corredor Interoceánico del Istmo de Tehuantepec (CIIT), Oaxaca. Integra datos de temperatura NASA POWER (2010–2025) con unidades económicas DENUE-INEGI (2020–2025) mediante interpolación IDW, tendencias Theil–Sen y una matriz de posicionamiento climático-económico. Desarrollado como extensión del procedimiento 3.3.2 del análisis geoespacial CIIT.
